## Overview

Essal Institute offers state-of-the-art training in computing. Our courses are divided into long-term state diploma programs and short-term professional bootcamps. All courses include extensive lab exercises. Schedules and instructor details are updated weekly on the [Student Portal](https://app.essal.institute).

---

## 1. State Diploma Programs (Long-Term)

### Technicien Supérieur (TS) in Software Engineering (Développement Web & Mobile)
- **Duration:** 30 Months (including a 6-month end-of-study graduation project/internship).
- **Accreditation:** Government-accredited TS degree.
- **Target Audience:** Baccalauréat holders or candidates with equivalent experience.

#### Module Structure
* **Semester 1:** Computer architecture, Algorithms and Logic, Basic HTML/CSS/JavaScript, Algebra, Technical English.
* **Semester 2:** Object-Oriented Programming (Java/C#), Database Design (SQL), Linux Fundamentals, Web Design Principles.
* **Semester 3:** Advanced Frameworks (PHP Laravel / React.js), Mobile App Development (React Native), API Design (REST), Software Testing.
* **Semester 4:** Cloud Computing basics, Git & Team Collaboration, project management, and start of Graduation Project.
* **Semester 5:** Mandatory 6-month Internship (Stage Pratique) and Thesis Defense.

### Technicien Supérieur (TS) in Computer Networks & Systems (Réseaux et Systèmes)
- **Duration:** 30 Months (including a 6-month internship).
- **Accreditation:** Government-accredited TS degree.
- **Alignment:** Aligns with Cisco CCNA and Microsoft Azure Administrator content.

#### Module Structure
* **Semester 1:** Intro to Networking, OSI Model, Subnetting, Basic PC Assembly, English for IT.
* **Semester 2:** Routing Protocols (OSPF, EIGRP), Switching and VLANs, Windows Server Administration (Active Directory, DNS, DHCP).
* **Semester 3:** Linux Server administration, Network Security (Firewalls, ACLs), Virtualization technologies (VMware ESXi), WAN technologies.
* **Semester 4:** Cloud Infrastructure Administration (Microsoft Azure), Network Monitoring (Nagios, Zabbix), Project Work.
* **Semester 5:** Mandatory 6-month Internship and Thesis Defense.

## 2. Professional Certification Bootcamps (Short-Term)

These accelerated tracks are designed for working professionals or university graduates seeking immediate skills acquisition:

### Full-Stack Web Development Bootcamp
* **Duration:** 6 Months (Weekend or Evening slots).
* **Stack:** Node.js, React, Express, MongoDB (MERN Stack).
* **Credential:** Essal Professional Certificate.
* **Portal Features:** Enrolled students get free access to GitHub Codespaces via our educational portal.

### Cybersecurity Fundamentals (Security+)
* **Duration:** 3 Months.
* **Focus:** Network vulnerabilities, threat identification, encryption techniques, and incident response.
* **Credential:** Prep for CompTIA Security+ certification.

### Database Administration (SQL Server & PostgreSQL)
* **Duration:** 2 Months.
* **Focus:** Database setup, query optimization, backup strategies, and replication.
* **Credential:** Essal Technical Certificate.
